ComfyUI + OpenWebUI

OpenWebUI 接入AI绘画

OpenWebUI设置

  1. 打开管理员面板，切换至设置–图像
  2. 打开图像生成（实验性）
  3. 填入 ComfyUI 基础地址，默认是 http://127.0.0.1:8188
  4. 上传从 ComfyUI 导出 API 格式的 workflow.json 文件.
  5. 填写ComfyUI Workflow 节点的序号。
  6. 设置默认模型、图片分辨率、步骤。

comfyui设置

  1. 到高级选项-网络设置，打开开放远程连接。
  2. 点击界面左上角工作流，选择导出(API)

使用

  1. 可以让大模型辅助生成提示词

    你是一位ComfyUI提示词助手，你的任务是根据我的文本生成提示词。提示词由普通常见的英语单词或短句构成，使用英文半角”,”做为分隔符。你要将我输入的中文文本翻译为英文再提取出提示词并对细节进行补充，并且在所有提示词开头默认加入以下提示词：best quality,masterpiece,ultra high res,incredibly absurdres,cinematic,highest quality,high resolution,cinematic lighting,

  2. 等待大模型生成提示词后，点击文本下方的生成图像

  3. 如果显存不够同时运行本地模型和ComfyUI，也可以直接编辑大模型生成的文本，自己写提示词。

ComfyUI 备份环境

  1. 在整合包中选择 高级选项-启动命令提示符
  2. 输入 pip freeze > requirements.txt
  3. 在根目录会生成requirements.txt，如果环境搞崩了就删除对应节点回退版本。
